# React Blog

A portfolio project — a blog browser built with React and Vite. Fetches users, posts, and comments from the [JSONPlaceholder](https://jsonplaceholder.typicode.com) API.

## Features

- Browse a list of users with live name filtering
- Click a user to view their posts
- Toggle comments on any post
- Custom `useFetch` hook with abort-on-unmount and error handling

## Stack

- React 19 + React Router 7
- Vite (SWC)

## Getting Started

```bash
npm install
npm run dev
```
